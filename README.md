# 课程管理系统后台系统（前后端分离系统）

这个后台项目用到的技术有:

* SpringBoot
* MybatisPlus
* Mysql
* Gradle
* log4j2和Druid连接池


## 特点

* 简单易用
* 跨域调用
* ...


## 使用方法

#### **推荐使用IntelliJ IDEA（或其他编译工具）**

将项目导入到**IntelliJ IDEA**中，然后利用gradle版本控制工具导入jar，导入sql，修改application.yml（换成自己的配置），启动主函数App.java即可。
